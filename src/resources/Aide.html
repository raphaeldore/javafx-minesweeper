Démineur : règles et principes de base

Objectif

Découvrez les cases vides en évitant les mines. Plus vite vous aurez terminé, meilleur sera votre score.

Plateau

Le Démineur offre un choix de trois plateaux standard de difficulté croissante.

  • Débutant : 81 cases, 10 mines

  • Intermédiaire : 256 cases, 40 mines

  • Expert : 480 cases, 99 mines

Comment jouer

Les règles du Démineur sont simples :

  • Si vous découvrez une mine, le jeu prend fin.

  • Si vous découvrez une case vide, vous pouvez continuer à jouer.

  • Si vous découvrez un chiffre, celui-ci indique le nombre de mines dissimulées dans les huit carrés qui l’entourent ;ces informations vous aident à déterminer quelles sont les cases voisines sur lesquelles vous pouvez cliquer sans risque.

Conseils et astuces

Marquez les mines. Si vous pensez qu’une case contient une mine, cliquez dessus avec le bouton droit. Un drapeau s’affiche sur la case. (En cas de doute, cliquez dessus à nouveau avec le bouton droit pour en faire un point d’interrogation.)

Apprenez à reconnaître les combinaisons numériques. Si trois cases à la suite indiquent 2-3-2, cela signifie qu’il y a probablement trois mines alignées à proximité de cette rangée. Si une case indique le chiffre 8, cela signifie qu’il y a une mine dans toutes les cases qui l’entourent.

Explorez les zones inexplorées. Vous ne savez pas trop où cliquer ensuite ? Tentez de défricher les zones inexplorées. Il vaut mieux cliquer au milieu de cases qui ne sont pas marquées plutôt que dans une zone où vous pensez qu’il y a des mines.
